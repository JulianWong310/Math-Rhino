## Practice Questions

**Q1:** Which of the following lines is the steepest?

A. $y = x$  
B. $y = -x$  
C. $y = 4x$  
D. $y = \frac{1}{4}x$

**Q2:** A proportional function passes through $(1, -5)$. What is its equation?

**Q3:** Compare $k_1$ and $k_2$ if the line $y = k_1x$ is steeper than $y = k_2x$ and both have $k > 0$.

**Q4:** Graph $y = -\frac{2}{3}x$ using the two-point method.

**Q5:** A proportional function passes through $(2, 8)$. Find its equation and state whether it is increasing or decreasing.

**Q6:** For the proportional function $y = (m - 5)x$, if the line passes through Quadrants II and IV, what must be true about $m$?

**Q7:** The function $y = (b^2 - 4)x^2 + 2(b - 2)x$ is a proportional function of $x$. Find the value of $b$ and the equation of the function.

**Q8:** For the proportional function $y = -7x$, complete the table and graph it.

| $x$ | -2 | 0 | 3 |
|---|---|---|---|
| $y$ | ? | ? | ? |

**Q9:** Two proportional functions are graphed: $y = k_1x$ and $y = k_2x$. If $k_1 = -4$ and $k_2 = -1$, which line is steeper?

**Q10:** For the proportional function $y = kx$, if $k < 0$ and $x_1 < x_2$, what is the relationship between $y_1$ and $y_2$?

**Q11:** The function $y = (|c| - 5)x^2 + 3(c - 5)x$ is a proportional function.

(a) Find the value of $c$.  
(b) Write the equation of the function.  
(c) Determine if the function is increasing or decreasing.  
(d) If $(x_1, y_1)$ and $(x_2, y_2)$ are on the graph and $x_1 > x_2$, compare $y_1$ and $y_2$.
