# 🏆 Functions & Domain

**Problem 1**
In the mathematical expression for the area of a square, $A = s^2$, which of the following is true?
* (A) $A$ is a constant.
* (B) $s$ is a constant.
* (C) $A$ is the independent variable.
* (D) $s$ is the independent variable.
* (E) The number $2$ is a variable.

**Problem 2**
Which of the following equations defines $y$ as a function of $x$?
* (A) $x^2 + y^2 = 25$
* (B) $x = |y| + 1$
* (C) $y = \frac{1}{x^2 + 1}$
* (D) $x = 3$
* (E) $y^2 = x$

**Problem 3**
What is the domain of the function $f(x) = \frac{100}{x-7}$?
* (A) All real numbers
* (B) $x > 7$
* (C) $x < 7$
* (D) $x \neq 7$
* (E) $x = 7$

**Problem 4**
Consider the function $g(x) = \sqrt{3x - 12}$. What is the smallest possible value in the domain?
* (A) 0
* (B) 3
* (C) 4
* (D) 12
* (E) No smallest value exists

**Problem 5**
Find the domain of the function $h(x) = (x-5)^0 + \frac{1}{x}$.
* (A) $x \neq 0$
* (B) $x \neq 5$
* (C) $x \neq 0$ and $x \neq 5$
* (D) $x > 0$
* (E) All real numbers

**Problem 6**
A "Number Converter" (function) follows these rules:
- Step 1: Multiply the input $x$ by 3.
- Step 2: Subtract 5 from the result.
If the output $y$ is 10, what was the input $x$?
* (A) 5
* (B) 15
* (C) 25
* (D) 35
* (E) 45

**Problem 7**
Which of the following values is NOT in the domain of $f(x) = \frac{\sqrt{x+1}}{x-2}$?
* (A) -1
* (B) 0
* (C) 1
* (D) 2
* (E) 3

**Problem 8**
If $f(x) = x^2 - 3x$, what is the value of $f(4)$?
* (A) 4
* (B) 7
* (C) 12
* (D) 16
* (E) 28

**Problem 9**
The domain of the function $f(x) = \sqrt{10-x}$ consists of all $x$ such that:
* (A) $x \geq 10$
* (B) $x \leq 10$
* (C) $x > 10$
* (D) $x \neq 10$
* (E) $x \geq 0$

**Problem 10**
Julian creates a special function where $y$ is the remainder when $x$ is divided by 4. If the domain is the set of integers $\{10, 11, 12, 13\}$, what is the range (值域) of this function?
* (A) $\{0, 1, 2, 3\}$
* (B) $\{1, 2, 3\}$
* (C) $\{2, 3, 0, 1\}$
* (D) $\{10, 11, 12, 13\}$
* (E) $\{2, 3, 4, 1\}$

---

# 🔑 Answer Key

1. **D** ($s$ is the input/independent variable; $A$ is the output; $2$ is a constant.)
2. **C** (For any $x$, $x^2+1$ is a unique value. In A, B, and E, one $x$ can have two $y$ values.)
3. **D** (The denominator $x-7 \neq 0$, so $x \neq 7$.)
4. **C** ($3x - 12 \geq 0 \implies 3x \geq 12 \implies x \geq 4$.)
5. **C** (Rule 1: Base $x-5 \neq 0$; Rule 2: Denominator $x \neq 0$.)
6. **A** ($3x - 5 = 10 \implies 3x = 15 \implies x = 5$.)
7. **D** (Denominator $x-2 \neq 0$ means $x$ cannot be 2.)
8. **A** ($f(4) = 4^2 - 3(4) = 16 - 12 = 4$.)
9. **B** ($10 - x \geq 0 \implies 10 \geq x$ or $x \leq 10$.)
10. **C** ($10\div4$ rem 2; $11\div4$ rem 3; $12\div4$ rem 0; $13\div4$ rem 1. The set is $\{2, 3, 0, 1\}$.)
