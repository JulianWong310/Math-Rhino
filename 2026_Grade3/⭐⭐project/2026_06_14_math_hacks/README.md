# Math Logic & Creative Challenges

Welcome to my math project! 

This project contains my collection of **"Fun Calculations"**—math problems that I've discovered can be solved using logical shortcuts and clever patterns instead of traditional long-form arithmetic. 

Inspired by these techniques, I have also created my own set of challenges to share with my classmates, teachers, and family.

## Project Highlights
- **Fun Calculations:** A collection of cool math tricks and shortcuts I've practiced.
- **Numbers and Riddles:** A series of math riddles and problems I designed myself, based on the patterns I learned.

## Learning Approach
Instead of just "grinding" through numbers, I focus on:
* **The Distributive Property:** Finding faster ways to multiply and add.
* **Decomposition:** Breaking big numbers into friendly, easier-to-manage parts.
* **Pattern Recognition:** Using logic to find the answers to complex-looking equations.
