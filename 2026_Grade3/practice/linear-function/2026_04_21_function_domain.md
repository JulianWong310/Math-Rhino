# 🎓 Math Lecture: Variables and the Logic of Functions


## 1. Constants vs. Variables 
In any mathematical process, we deal with two types of quantities:
* **Constant:** A value that **never changes**. (e.g., the number $2$ or $\pi$).
* **Variable:** A value that **can change** or take on different numbers.

> **Example:** In the circle formula $C = 2\pi r$:
> * $2$ and $\pi$ are the **Constants**.
> * $C$ and $r$ are the **Variables**.

---

## 2. What is a Function? 
A **Function** is a relationship between two variables. It acts like a "Number Converter."

* **Independent Variable:** The "Input" (usually $x$).
* **Dependent Variable:** The "Output" (usually $y$), which depends on $x$.

**The Golden Rule:**
For every input ($x$), there must be **exactly one** output ($y$). 
* If one $x$ points to two different $y$ values, it is **not** a function.

---

## 3. The Three Elements of a Function 
1.  **Domain:** The set of all allowed input values ($x$).
2.  **Correspondence:** The rule or formula (e.g., $f(x) = 2x$).
3.  **Range:** The set of all resulting output values ($y$).

---

## 4. How to Find the Domain
When looking for the **Domain**, you must follow these "Safety Rules":

1.  **Fractions:** The **Denominator** cannot be zero.
    * *Example:* $y = \frac{1}{x-3} \implies x-3 \neq 0 \implies$ **Domain:** $x \neq 3$.
2.  **Square Roots:** The value inside must be **non-negative** (greater than or equal to zero).
    * *Example:* $y = \sqrt{x-2} \implies x-2 \geq 0 \implies$ **Domain:** $x \geq 2$.
3.  **Zero Power:** The **Base** cannot be zero.
    * *Example:* $y = (x+1)^0 \implies x+1 \neq 0 \implies$ **Domain:** $x \neq -1$.

# 📝 Practice Problems

**Problem 1: Identifying Variables**
In the formula $y = 5x + 3$:
* Which is the **Independent Variable**?
* Which is the **Dependent Variable**?
* Which are the **Constants**?

**Problem 2: Function Check**
Does the equation $x = y^2$ represent $y$ as a **Function (函数)** of $x$? Why or why not?

**Problem 3: Domain Search**
Find the **Domain** for: 
$$y = \frac{\sqrt{x-5}}{x-10}$$

---

# 🔑 Answers and Logic

1. **Variables & Constants**
   * **Independent Variable:** $x$
   * **Dependent Variable:** $y$
   * **Constants:** $5$ and $3$

2. **Function Check**
   * **Answer:** **No**. 
   * **Logic:** If $x = 4$, $y$ could be $2$ or $-2$. Because one input has two outputs, it fails the "Golden Rule" of functions.

3. **Domain Search**
   * **Rule 1 (Root):** $x-5 \geq 0 \implies x \geq 5$.
   * **Rule 2 (Fraction):** $x-10 \neq 0 \implies x \neq 10$.
   * **Combined Answer:** **$x \geq 5$ and $x \neq 10$**.
