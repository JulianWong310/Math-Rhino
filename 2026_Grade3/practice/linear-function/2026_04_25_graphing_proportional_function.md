# Graphing Proportional Functions

## Example 1

Graph the following functions on the same coordinate plane:

$$ y = x, \quad y = -x, \quad y = 2x, \quad y = -2x $$

### Table of Values

| $x$ | ... | -2 | -1 | 0 | 1 | 2 | ... |
|---|---|---|---|---|---|---|---|
| $y = x$ | ... | -2 | -1 | 0 | 1 | 2 | ... |
| $y = -x$ | ... | 2 | 1 | 0 | -1 | -2 | ... |
| $y = 2x$ | ... | -4 | -2 | 0 | 2 | 4 | ... |
| $y = -2x$ | ... | 4 | 2 | 0 | -2 | -4 | ... |

### Key Observations

- All graphs are straight lines passing through the origin $(0,0)$.
- When $k > 0$, the line rises (increases) as $x$ increases.
- When $k < 0$, the line falls (decreases) as $x$ increases.
- $y = x$ and $y = 2x$ are in Quadrants I and III.
- $y = -x$ and $y = -2x$ are in Quadrants II and IV.
- The larger the value of $|k|$, the steeper the line.

### Two-Point Method for Graphing $y = kx$

Since the graph passes through $(0,0)$, you only need one more point:

1. Plot $(0,0)$.
2. Plot $(1, k)$.
3. Draw a straight line through both points.

## Example 2

Given the functions:

$$ y = x, \quad y = -2x, \quad y = \frac{1}{2}x, \quad y = 3x $$

### (1) Graph the functions on the same coordinate plane.

### (2) Discovery

As $|k|$ increases, the line becomes **steeper**.

### (3) Application

The graph shows two proportional functions:

$$ y = k_1 x \quad \text{and} \quad y = k_2 x $$

From the graph: $k_2 < k_1$

### Table of Values

| $x$ | ... | 0 | 1 | ... |
|---|---|---|---|---|
| $y = x$ | ... | 0 | 1 | ... |
| $y = -2x$ | ... | 0 | -2 | ... |
| $y = \frac{1}{2}x$ | ... | 0 | $\frac{1}{2}$ | ... |
| $y = 3x$ | ... | 0 | 3 | ... |

### Diagram Description

- The line for $y = k_1 x$ is steeper than the line for $y = k_2 x$.
- Both lines pass through the origin.

## Example 3

The function below is a proportional function of $x$:

$$ y = (|a| - 3)x^2 + 2(a - 3)x $$

### (1) Find the equation of the proportional function.

**Step 1:** For the function to be proportional, the $x^2$ term must disappear (coefficient = 0).

$$ |a| - 3 = 0 \implies |a| = 3 \implies a = 3 \quad \text{or} \quad a = -3 $$

**Step 2:** The coefficient of $x$ must not be zero ($k \neq 0$).

$$ 2(a - 3) \neq 0 \implies a - 3 \neq 0 \implies a \neq 3 $$

**Step 3:** Combine the conditions.

$$ a = -3 \quad (\text{since } a = 3 \text{ is excluded}) $$

**Step 4:** Substitute $a = -3$ into the function.

$$ y = (|-3| - 3)x^2 + 2(-3 - 3)x $$
$$ y = (3 - 3)x^2 + 2(-6)x $$
$$ y = 0 \cdot x^2 - 12x $$
$$ y = -12x $$

**Answer:** $y = -12x$

### (2) Graph the function.

Using the two-point method:
- Plot $(0, 0)$
- Plot $(1, -12)$
- Draw a straight line through both points.

The line passes through Quadrants II and IV and is very steep because $|k| = 12$.

### (3) Compare $y_1$ and $y_2$ given $x_1 < x_2$.

Since $k = -12 < 0$, the function is **decreasing**. This means:

- As $x$ increases, $y$ decreases.
- Therefore, if $x_1 < x_2$, then $y_1 > y_2$.

**Answer:** $y_1 > y_2$

## Summary of Key Concepts

| Concept | Description                                  |
|---------|----------------------------------------------|
| Form | $y = kx$, $k \neq 0$                         |
| Graph | Straight line through $(0,0)$                |
| $k > 0$ | Line rises (Quadrants I and III)             |
| $k < 0$ | Line falls (Quadrants II and IV)             |
| Steepness | Increases as absolute value of $k$ increases |
| Graphing method | Two points: $(0,0)$ and $(1, k)$             |
| Increasing/decreasing | $k > 0$: increasing; $k < 0$: decreasing     |
