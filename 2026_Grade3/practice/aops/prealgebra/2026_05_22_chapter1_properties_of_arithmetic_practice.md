## Properties Of Arithmetic

**Part 1: Basic Properties of Arithmetic**

Q1: Which of the following equations correctly represents the commutative property of addition?
- A. $(a + b) + c = a + (b + c)$
- B. $a + b = b + a$
- C. $a + 0 = a$
- D. $a(b + c) = ab + ac$

**Part 6: Arithmetic Warnings (Common Pitfalls)**

Q38: Which of the following statements about subtraction is true?

- B. Subtraction is associative ($a - (b - c) = (a - b) - c$).
- C. Subtraction is neither commutative nor associative.
- D. Subtraction is not commutative, but it is associative.

---

Q39: In arithmetic, what is the rule regarding dividing by zero (e.g., $a \div 0$ or $\frac{a}{0}$)?
- A. The result is $0$.
- B. The result is $a$.
- C. Division by zero is undefined.
- D. The result is $1$.

---

Q40: Which of the following statements about division is true?
- A. Division is commutative ($a \div b = b \div a$).
- B. Division is associative ($a \div (b \div c) = (a \div b) \div c$).
- C. Division is both commutative and associative.
- D. Division is neither commutative nor associative.

---

Q41: We know that $(a + b) \div c = a \div c + b \div c$. If we reverse the order, does $a \div (b + c)$ equal $a \div b + a \div c$?
- A. Yes, because division distributes completely over addition.
- B. No, $a \div (b + c)$ is not necessarily equal to $a \div b + a \div c$.
- C. Yes, they are only unequal if $a = 0$.
- D. Yes, these two expressions mean the exact same thing.
