# 🗝️ Master Answer Key 

| Level | Answer | Core Concept | The Logic Test (The "Why") |
| :--- | :--- | :--- | :--- |
| **1A (Race)** | **60** | **Permutation** | **Order Matters:** Swapping 1st and 2nd place creates a different result. Calculation: $5 \times 4 \times 3$. |
| **1B (Pizza)** | **10** | **Combination** | **Order Doesn't Matter:** Picking Alex then Ben is the same group as Ben then Alex. Calculation: $\frac{5 \times 4 \times 3}{3 \times 2 \times 1}$. |
| **2A (Lock)** | **125** | **Repetition Allowed** | **Replacement:** You can use the same digit multiple times. Each slot has 5 choices. Calculation: $5 \times 5 \times 5 = 5^3$. |
| **2B (ID)** | **60** | **No Repetition** | **Counting Down:** Once a digit is used, it’s gone. Choices go 5, then 4, then 3. Calculation: $5 \times 4 \times 3$. |
| **3 (Code)** | **12** | **Case Study** | **Constraints First:** You must handle the "Even" rule first. The last digit must be 2 or 4. Case 1 (ends in 2) + Case 2 (ends in 4) = $6 + 6 = 12$. |
| **4 (Captain)** | **60** | **Mixed Logic** | **Multi-Step:** Pick 1 specific leader (6 choices), THEN pick an unordered group of 2 from the remaining 5 ($\binom{5}{2} = 10$). Calculation: $6 \times 10$. |
| **5 (MATH)** | **24 / 6** | **Factorials** | **Fixed Positions:** Total permutations is $4! = 24$. If 'M' is locked in 1st place, you only arrange the remaining 3 letters ($3! = 6$). |
